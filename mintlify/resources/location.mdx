The `client.location` resource lets you look up the current location of any entity (character, ship, vehicle, etc.).

### location.get()

Get an entity's current location.
Returns the `Location` object directly — not wrapped in a `Page`.

<ParamField path="entityType" type="string" required>
  The entityType parameter.
</ParamField>

<ParamField path="uid" type="string" required>
  The uid parameter.
</ParamField>

```typescript
const location = await client.location.get({ entityType: 'character', uid: '1:12345' });
console.log(location); // access properties directly, not location.data
```
