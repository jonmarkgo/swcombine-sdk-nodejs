## Galactic News Service (GNS)

### news.gns.get()

Get a specific GNS news item by numeric ID.
Returns the `NewsItem` object directly â€” not wrapped in a `Page`.
Author and faction are normalized to object references with `value`.
If the API returns an array, the SDK returns the first normalized item.

<ParamField path="id" type="string \| number" required>
  News item ID from list results (`attributes.id`)
</ParamField>

```typescript
const post = await client.news.gns.get({ id: 49108 });
console.log(post.title);          // access properties directly, not post.data
console.log(post.author.value);
console.log(post.faction.value);
```

### news.gns.list()

List GNS news items (paginated with optional filtering)
Returns a Page of headline metadata entries (`attributes.id`, `attributes.href`, `attributes.title`, `value`).
Use `page.data` for the current page items, `page.hasMore` / `page.getNextPage()` for pagination,
or `for await (const item of page)` to iterate through all pages automatically.

</ParamField>

<ParamField path="end_date" type="**end\_date**: number">

</ParamField>

<ParamField path="faction" type="string">
  Faction name to filter by (GNS only)

  Faction type to filter by (GNS only)
</ParamField>

<ParamField path="item_count" type="**item\_count**: number" default="50">
