The `client.inventory` resource lets you list and inspect entities — ships, vehicles, facilities, items, NPCs, and more — that belong to your character or faction.

### inventory.get()

Get inventory summary by UID.
Returns the inventory summary object directly — not wrapped in a `Page`.

<ParamField path="uid" type="string" required>
  The uid parameter.
</ParamField>

```typescript
const overview = await client.inventory.get({ uid: '1:12345' });
console.log(overview); // access properties directly, not overview.data
```

### inventory.entities.list()

List entities in inventory (paginated with optional filtering)
Supports filtering by various entity properties. Filter arrays must have matching lengths.

<ParamField path="assignType" type="InventoryAssignType" required>
  Assignment type: 'owner', 'commander', or 'pilot'
</ParamField>

<ParamField path="entityType" type="T" required>
  Entity type: 'ships', 'vehicles', 'stations', 'cities', 'facilities', 'planets', 'items', 'npcs', 'droids', 'creatures', or 'materials'
</ParamField>

<ParamField path="filter_inclusion" type="**filter\_inclusion**: InventoryFilterInclusion[]">
  Whether each filter should include or exclude matches. Default: 'includes'
</ParamField>

<ParamField path="filter_type" type="**filter\_type**: InventoryFilterType[]">
  Filter types to apply to the query
</ParamField>

<ParamField path="filter_value" type="**filter\_value**: string[]">
  Values corresponding to each filter type
</ParamField>

<ParamField path="item_count" type="**item\_count**: number" default="50">
  Number of items to retrieve. Default: 50, Max: 200
</ParamField>

<ParamField path="pageDelay" type="number">
  Milliseconds to wait before fetching each subsequent page. Helps avoid rate limits during auto-pagination.
</ParamField>

<ParamField path="start_index" type="**start\_index**: number" default="1">
  Starting position for pagination (1-based). Default: 1
</ParamField>

<ParamField path="uid" type="string" required>
  The uid parameter.
</ParamField>

```typescript
const entities = await client.inventory.entities.list({ uid: '1:12345', entityType: 'vehicle', assignType: 'pilot' });
// Fetch up to 200 entities at once
const moreEntities = await client.inventory.entities.list({ uid: '1:12345', entityType: 'vehicle', assignType: 'pilot', start_index: 1, item_count: 200 });
// Filter by multiple criteria
const multiFiltered = await client.inventory.entities.list({
  uid: '1:12345',
  entityType: 'ships',
  assignType: 'owner',
  filter_type: ['class', 'powered'],
  filter_value: ['Fighter', '1'],
  filter_inclusion: ['includes', 'includes']
});
```
