The `client.market` resource provides access to NPC vendor listings and marketplace data.

## Vendors

### market.vendors.get()

Get a specific vendor by UID.
Returns the `Vendor` object directly — not wrapped in a `Page`.

<ParamField path="uid" type="string" required>
  The uid parameter.
</ParamField>

```typescript
const vendor = await client.market.vendors.get({ uid: 'vendor-uid' });
console.log(vendor.name); // access properties directly, not vendor.data
```

### market.vendors.list()

List all public vendors (paginated)

<ParamField path="item_count" type="number">
  The item_count parameter.
</ParamField>

<ParamField path="pageDelay" type="number">
  The pageDelay parameter.
</ParamField>

<ParamField path="start_index" type="number">
  The start_index parameter.
</ParamField>

```typescript
const vendors = await client.market.vendors.list();
const moreVendors = await client.market.vendors.list({ start_index: 51, item_count: 50 });
```
